hexo使用

2015-01-06 16:29:27

1.安装Node.js

wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.20.0/install.sh | bash

2.安装Node.js

nvm install stable

3.安装hexo

npm install hexo -g

hexo init blog

cd blog

npm install

hexo server

4.使用

将_config.yml修改如下：

deploy:

type: github

repository: git@github.com:xuefliang/xuefliang.github.io.git

branch: master

发布文章

hexo new Test 修改source-_posts 下的Test.md文件

hexo clean

hexo generate

hexo deploy
